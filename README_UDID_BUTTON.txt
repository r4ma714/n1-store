و link ـەکە بکاتە:
http://localhost:3000/udid

دانان:
1. ZIP ـەکە Extract بکە.
2. فۆڵدەری scripts copy بکە.
3. بیخە ناو:
   C:\Users\rama7\Documents\n1
4. لە فۆڵدەری n1 دوو کلیک لەسەر:
   change-home-telegram-to-udid.ps1
   بکە
   یان لە PowerShell:
   .\scripts\change-home-telegram-to-udid.js
5. Browser refresh بکە: Ctrl + R
