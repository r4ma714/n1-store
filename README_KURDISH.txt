N1 UDID 404 Fix

ئەم ZIP ـە 404 ـی /admin/udids چاک دەکات و پەڕەی UDID زیاد دەکات.

دانان:
1. ZIP extract بکە.
2. فۆڵدەری app copy بکە.
3. بیخە ناو C:\Users\rama7\Documents\n1
4. Replace بکە.
5. ئەگەر server کراوەیە، Ctrl + C بکە، پاشان npm run dev.
6. ئەم لینکە تاقی بکە:
   http://localhost:3000/udid
   http://localhost:3000/admin/udids

گرنگ:
- بۆ کارکردنی UDID لە iPhone باشترینە دوای deploy بە HTTPS domain تاقی بکرێت.
- لوکال بۆ بینینی design و route کار دەکات.
